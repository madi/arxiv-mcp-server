arXiv MCP Server

Search and analyze academic papers directly from Claude, ChatGPT, or any MCP-compatible AI assistant.

Get Started View on GitHub

Available Tools

Three tools that give your AI assistant full access to arXiv's academic corpus.

search_papers

Search papers by keywords, topic, or author. Returns titles, authors, abstracts, and links — sorted by relevance or date.

get_paper

Get full details of any paper by its arXiv ID. Includes abstract, all authors, categories, and PDF link.

search_by_category

Browse papers by arXiv category — cs.AI, cs.LG, stat.ML, and hundreds more. Filter with optional keywords.

Get Started

Choose the setup that works for you.

Hosted Endpoint

No install needed

Add this URL to Claude.ai (Settings → Integrations → Add MCP Server) or any MCP client:

https://arxiv-mcp-server.dileomargherita.workers.dev/mcp

Claude Desktop

Local

Add to your Claude Desktop config file (claude_desktop_config.json):

{
  "mcpServers": {
    "arxiv": {
      "command": "npx",
      "args": ["arxiv-mcp-server"]
    }
  }
}

VS Code / Cursor

Local

Add to your MCP settings:

{
  "mcpServers": {
    "arxiv": {
      "command": "npx",
      "args": ["arxiv-mcp-server"]
    }
  }
}

Self-hosted (Cloudflare Workers)

Deploy your own

Clone, build, and deploy to your own Cloudflare account:

git clone https://github.com/madi/arxiv-mcp-server
cd arxiv-mcp-server
npm install
wrangler login
wrangler deploy
